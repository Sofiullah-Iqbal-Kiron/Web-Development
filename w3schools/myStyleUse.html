Color Discussion

HTML/CSS supports 140 color name. You can get total list of the colors from w3schools color reference cite. Although, there are many color processing syntax as:-

  1. Color Name. By default, I use this.
  2. RGB(int red, int green, int blue) in [0, 255] interval. If value of all parameters are same than it will produce different sets of gray. All 255 is white and all 0 is black.
  3. RGBA(int red[0, 255], int green[0, 255], int blue[0, 255], int absolute[0, 1])
