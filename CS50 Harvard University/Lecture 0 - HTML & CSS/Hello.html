  1. h1, h2, h3, h4, h5 and h6: Headings, Biggest heading as possible is h1, then h2, h3 and all the way down to h6.

  4. ul: Unordered list
  5. img: Image
  6. a: Anchor, link

What is Java?

Java is a general-purpose class-based, object-oriented programming language designed for having lesser implementation dependencies. It is a computing platform for application development. Java is faster than python, secure and reliable. It is widely used for developing Java applicatinos in laptops, data centers, game consoles, secure corporate applications.
